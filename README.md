## Tecnologías utilizadas

- **HTML5**: estructura de la página (`index.html`), con etiquetas semánticas (`header`, `main`, `nav`, `section`, `footer`).
- **CSS3**: hoja de estilos propia (`style.css`) para personalizar la apariencia por encima de Bootstrap.
- **Bootstrap 5.3.8**: framework CSS cargado vía CDN (jsDelivr), usado para el estilo de la barra de navegación (`nav-link`) y los botones (`btn btn-primary`).
